# Lists & Boxes in HTML

## Lists

HTML provides us with three different ways to display content:

- Ordered lists (ol)
  - Unordered lists (ul)
    *- Definition lists (?)*

    Each list type contains items known as (li), as sees on the table below:

    Ordered Lists | Unordered Lists
    --------------|-----------------
< ol >
< li > TEXT < / li >
< li > TEXT2 < / li > | < ul >
< / ol >
< li > TEXT < / li >
< li > TEXT2 < / li >
< / ol >
